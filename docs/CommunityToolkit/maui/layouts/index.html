.NET MAUI Community Toolkit Layouts

The .NET MAUI Community Toolkit extends the list of .NET MAUI layouts. Here are the layouts provided by the toolkit:

View Description
DockLayout The DockLayout is a layout where views can be docked to the sides (top, left, right, bottom) of the layout container.
StateContainer The StateContainer bindable properties enable any Layout derived element to become state-aware.
UniformItemsLayout The UniformItemsLayout is a layout where all rows and columns have the same size.
